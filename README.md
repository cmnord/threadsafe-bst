# Threadsafe BST

Threadsafe Binary Search Tree written in Rust. Nodes are wrapped by
`Arc<Mutex<>>`.

Implementation based on Chapter 15 of [The Complete Rust Programming Reference Guide][TCRPRG] red-black tree and binary search tree.
